# ls(현재 폴더의 상태보기)

# ls

현재 폴더 안에 있는 폴더, 파일을 보여준다.

```bash
ls
```

---

## ls -l

더 풍부한 정보(용량, 소유자, 생성시간 등등)를 표시해 준다.

```bash
ls -l
```

tip) 파일, 폴더명 앞에 -가 붙으면 파일이고 d가 붙으면 폴더이다.

## ls -a

숨긴 파일도 출력해준다.

```bash
ls -a  
```

```bash
ls -al
```

---

# ls -R

모든 폴더 아래에 있는 파일,

 폴더도 다 보여준다 

```bash
ls -R
```

---

## ls -S

파일을 용량이 큰 기준으로  정렬한다.

```bash
ls -S
```
