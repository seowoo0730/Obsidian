## and `&&`

**앞, 뒤 모든 조건이 True** 여야 True

```JavaScript
console.log(2 == 2 && 7 > 5); // true
```

## or `||`

**앞, 뒤 모든 조건 중 하나라도 True**라면 True

```JavaScript
console.log(2 == 2 || 7 < 5); // true
```

## not `!`

```JavaScript
console.log(!true); // false
```

### 동등

`==`

```JavaScript
console.log(10 == `10`); //true
```

```JavaScript
alert(null == undefined); //true
```

### 부등

`!=`

```JavaScript
console.log(10 != `10`); //false
```

### 일치

`===`

```JavaScript
console.log(10 === `10`); //false
```

```JavaScript
alert(null === undefined); //false
```

### 불일치

`!==`

```JavaScript
console.log(10 !== 10); //false
```
