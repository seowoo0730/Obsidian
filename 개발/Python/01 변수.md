## 변수 선언하기

`var`, `let`, `const`와 같은 **변수 선언 키워드를 사용하지 않고** 변수를 선언하는 것

```run-python
animal = '강아지'
name = '몽이'
age = 3
is_adult = age >= 3

print('우리집 강아지 이름은 ' + name + '입니다')
print(name + '는 ' + str(age) + '살입니다')
print(name, '는 어른일까요? ', is_adult) 
'''우리 집 강아지 이름은 몽이입니다
몽이는 3살입니다
몽이 는 어른일까요?  True'''
```
