### `random`

```Python
from random import *
print(random())  # 0.0 ~ 1.0 사이의 값
```

```Python
from random import *
print(int(random()*100))  # 0 ~ 100 사이의 값
```

### `randrange`

```Python
from random import *
print(randrange(1, 46))  # 0 ~ 46미만의 값
```

### `randint`

```Python
from random import *
print(randint(1, 45))  # 0 ~ 45 이하의 값
```
