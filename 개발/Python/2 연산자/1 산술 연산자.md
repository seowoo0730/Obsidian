### 덧셈

`+`

```Python
print(1 + 1);
```

```Python
print('안녕하세요' + ' 이서우입니다.'); # 안녕하세요 이서우입니다.
```

### 뺄셈

`-`

```Python
print(3 - 1); # 2
```

### 곱셈

`*`

```Python
print(3 * 2); # 6
```

### 나눗셈

`/`

```Python
print(4 / 2); # 2
```

### 제곱

`**`

```Python
print(2**3) # 8
```

### 몫

`//`

```Python
print(5 // 2) # 2
```

### 나머지

`%`

```Python
print(9 % 2); # 1
```

---

### `abs`(절댓값)

```Python
print(abs(-5)) # 5
```

### `pow`(제곱)

```Python
print(pow(-5, 2)) # 25
```

### `max`(최댓값)

```Python
print(min(abs(-5), 2))  # 2
```

### `min`(최솟값)

```Python
print(max(abs(-5), 2))  # 5
```

### `round`반올림

```Python
print(round(3.141592))  # 3
```
