## `and`

**앞, 뒤 모든 조건이 True** 여야 True

```JavaScript
print((2 == 2) and (7 > 5)) # True
```

```Python
print((2 == 2) && (7 > 5)) # True
```

## `or`

**앞, 뒤 모든 조건 중 하나라도 True**라면 True

```Python
print((2 == 2) or (7 < 5)) # True
```

```Python
print((2 == 2) | (7 < 5))  # True
```

### 일치

`==`

```Python
print(10 == 10); # True
```

### 불치

`!=`

```Python
print(10 != 10); # False
```

### 초과

`>`

```Python
print(20 > 10); # True
```

### 미만

`<`

```Python
print(20 < 10); # False
```

### 이상

`>=`

```Python
print(10 >= 10); # True
```

### 이하

`<=`

```Python
print(10 <= 20); # True
```
